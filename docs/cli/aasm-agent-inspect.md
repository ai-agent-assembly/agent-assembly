# `aasm agent inspect`

<a id="cmd-aasm-agent-inspect"></a>

Show detailed information about a specific agent

## Synopsis

```text
Usage: aasm agent inspect <AGENT_ID>
```

## Options

| Option | Value | Description |
|--------|-------|-------------|
| `<AGENT_ID>` | `<AGENT_ID>` | Hex-encoded agent UUID to inspect *(required)* |
