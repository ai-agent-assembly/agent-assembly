# `aasm completion`

<a id="cmd-aasm-completion"></a>

Generate shell completion scripts

## Synopsis

```text
Usage: aasm completion <SHELL>
```

## Options

| Option | Value | Description |
|--------|-------|-------------|
| `<SHELL>` | `<SHELL>` (`bash`, `elvish`, `fish`, `powershell`, `zsh`) | Shell to generate completions for *(required)* |
