# `aasm agent resume`

<a id="cmd-aasm-agent-resume"></a>

Resume a suspended agent

## Synopsis

```text
Usage: aasm agent resume <AGENT_ID>
```

## Options

| Option | Value | Description |
|--------|-------|-------------|
| `<AGENT_ID>` | `<AGENT_ID>` | Hex-encoded agent UUID to resume *(required)* |
