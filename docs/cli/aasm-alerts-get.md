# `aasm alerts get`

<a id="cmd-aasm-alerts-get"></a>

Show full detail for a specific alert

## Synopsis

```text
Usage: aasm alerts get <ALERT_ID>
```

## Options

| Option | Value | Description |
|--------|-------|-------------|
| `<ALERT_ID>` | `<ALERT_ID>` | Alert ID to inspect *(required)* |
